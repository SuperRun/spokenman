课程概述：

本课程主要讲硬笔书法的基本笔顺，硬笔书法的结构，硬笔书法的握笔姿势，硬笔书法偏旁部首，硬笔字示范等。 讲师田舍之，国学专家、汉字文化专家、中国书法家协会会员、中国青少年传统文化教育研究院高级研究员。 大家在学习的过程中有什么疑问，可添加小曰微信咨询：ziyuevideo666 欢迎大家加入优贝壳习字交流QQ群：231256952 另外田老师还在网易云课堂，推出了更加系统性的课程《跟舍之学写字》，该课程不局限于写字或书法，同时从国学、人生、修行、静心等方面进行延伸，深入浅出，通俗易懂。 《跟舍之学写字》是这套免费课程的升级进阶版本，可点击下图按钮了解该课程。
